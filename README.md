# Snow Bros

A Frozen-inspired remake of the classic 1990 arcade game, built with C++ and SFML 3.0.0 for our Object-Oriented Programming course. Trap enemies in snowballs, battle bosses, collect gems, and team up with a friend in local co-op.

## Features 

-  Single-player and local 2-player gameplay
-  Multiple enemy types and boss battles
-  Random Star Levels with power-up choices
-  Gems, upgrades, and an in-game shop
-  Persistent saves and leaderboard
-  Remappable control keys
-  10 levels with increasing difficulty

## Tech Stack 
- C++17
- SFML 3.0.0
- Visual Studio

## How to Run 

1. Clone the repository:

```bash
git clone https://github.com/meowryam/SnowBros.git
```

2. Open SnowBros.sln in Visual Studio.

3. Make sure SFML 3.0.0 is installed and configured correctly.

4. Build and run the project.

 Keep the assets folder in its original location for textures, fonts, and audio to load properly.
